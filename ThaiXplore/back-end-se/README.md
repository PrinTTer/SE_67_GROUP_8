## Installation & Run

```bash
git clone https://github.com/PrinTTer/SE_67_GROUP_8.git
cd ThaiXplore/back-end-se
npm install
npm start
```
## Installation Issue

- If you encounter any problems with nodemon after installing npm, try reinstalling nodemon.

```bash

npm i nodemon

```
